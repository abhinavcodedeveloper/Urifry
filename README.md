## Installation

### Prerequisites

- Node.js
- npm (Node Package Manager)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/urlyf.git

1. Go to the Directory:
   ```bash
    cd urlyf

1. Install the dependancies:
   ```bash
    npm i

1. Run the dev:
   ```bash
    npm run dev
